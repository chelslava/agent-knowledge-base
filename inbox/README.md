# 📥 Inbox — Входящие

Сырые входные данные, черновики, недооформленные мысли, вырезки. Временная зона.

## Правила

1. Сюда складывается **всё непонятое/неразобранное**.
2. При разборе (см. правило 6 в [[knowledge-rules]]):
   - определи тип и теги по [[taxonomy]];
   - выбери место: `shared/` (общее) или `projects/<name>/` (проектное);
   - переоформи по шаблону из `templates/`;
   - проставь `[[ссылки]]`;
   - удали исходник из `inbox/`.
3. Перед оформлением — **семантический поиск**, чтобы не плодить дубликаты.

> Inbox не должен накапливаться бесконечно — регулярно разбирай его.
