# 🌐 Shared — Общие переиспользуемые знания

**Источник общих знаний** для всех проектов. Сюда выносится всё, что не привязано
к одному проекту: паттерны, рецепты, гайды, чек-листы, подводные камни.

> Правило: проекты **ссылаются** на `shared/` через `[[...]]`, но **не копируют**
> его содержимое (правило 4 в [[AGENTS]], принцип SSOT в [[knowledge-rules]]).

## Рекомендуемая структура

```
shared/
├── patterns/     # архитектурные и кодовые паттерны   (kind/pattern)
├── recipes/      # пошаговые рецепты решения задач     (kind/recipe)
├── guides/       # гайды и руководства                 (kind/guide)
├── checklists/   # чек-листы                           (kind/checklist)
└── gotchas/      # подводные камни и грабли            (kind/gotcha)
```

Папки создаются по мере наполнения. Каждый документ — из шаблона
[[wiki-note]] с указанием `type/shared` и соответствующего `kind/*`.

## Когда выносить знание в shared

- Решение/паттерн пригодилось во **втором** проекте → выноси в `shared/`,
  в проекте оставь ссылку.
- Знание обобщаемо и не содержит проектных секретов.
